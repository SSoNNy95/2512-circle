# 원의 넓이 탐구하기

초등학교 6학년 학생들을 위한 인터랙티브 웹 애플리케이션으로, 원을 분할하고 재배열하는 과정을 통해 원의 넓이 공식을 스스로 발견하고 도출할 수 있도록 돕습니다.

## 주요 기능

### 1. 원 분할 및 시각화
- 슬라이더를 통해 원을 4, 8, 16, 32, 64, 128등분으로 분할
- 각 부채꼴 조각을 서로 다른 색상으로 구분하여 시각적으로 명확하게 표시
- 분할선이 명확하게 보이도록 처리

### 2. 드래그 앤 드롭 재배열
- 각 부채꼴 조각을 마우스/터치로 개별적으로 드래그하여 이동 가능
- 자유롭게 배치 가능한 "자유 모드"와 가이드라인에 스냅되는 "정렬 모드" 토글 기능

### 3. 자동 애니메이션
- "최적 배열 보기" 버튼: 부채꼴 조각들이 자동으로 평행사변형/직사각형 모양으로 부드럽게 이동
- "원래대로" 버튼: 조각들이 다시 원 형태로 돌아가는 애니메이션

### 4. 측정 도구
- 재배열된 도형의 가로 길이 측정 (원주의 절반)
- 재배열된 도형의 세로 길이 측정 (반지름)
- 각 측정값과 공식 표현을 함께 표시

### 5. 학습 단계별 UI
- **1단계: 탐구 모드** - 원 분할 및 자유 배열
- **2단계: 발견 모드** - 최적 배열 애니메이션 및 패턴 발견
- **3단계: 공식 도출 모드** - 측정 도구를 통한 공식 도출
- **4단계: 적용 모드** - 실제 사물 사진으로 넓이 계산

### 6. 구글 폼 연동
- 각 단계별 학습 활동 완료 시 타임스탬프 기록
- 학습 결과 및 측정 데이터 제출

## 기술 스택

- **React 18** + **TypeScript**
- **Framer Motion** - 애니메이션
- **@dnd-kit** - 드래그 앤 드롭
- **Tailwind CSS** - 스타일링
- **Vite** - 빌드 도구

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 사용 방법

1. **탐구 모드**: 슬라이더로 원을 나누고, 조각들을 드래그하여 자유롭게 배치해보세요.
2. **발견 모드**: "최적 배열 보기" 버튼을 눌러 조각들이 어떻게 배열되는지 관찰하세요.
3. **공식 도출 모드**: 측정 도구를 통해 가로와 세로 길이를 확인하고 공식을 도출해보세요.
4. **적용 모드**: 실제 원 모양 사물의 사진을 업로드하고 지름을 측정하여 넓이를 계산해보세요.
